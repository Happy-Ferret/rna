========================================
Welcome to RNA documentation!
========================================

This documentation explains how to install and use RNA, Mozilla's 
release-notes publishing application.


About RNA
-------------

RNA is an application in the `Nucleus <http://rtfd.org/nucleus>`_ publishing 
platform. It provides an interface for managing the content of release 
notes intended for publication on www.mozilla.org.
